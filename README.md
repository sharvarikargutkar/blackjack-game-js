# 🃏 Blackjack Game

A simple and interactive Blackjack game built using JavaScript.
This project demonstrates core programming concepts like game logic, DOM manipulation, and dynamic UI updates.

---

## 🚀 Features

* 🎮 Start a new Blackjack game
* 🃏 Draw new cards
* 🧮 Automatic sum calculation
* 💰 Chip system (win/lose tracking)
* 🧑 Player name input
* 📜 Game rules display
* 🔄 Restart game functionality
* 🎯 Real-time game status updates (Win / Lose / Continue)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ⚙️ How to Play

1. Enter your name to start the game
2. Click **Start Game** to receive two cards
3. Click **New Card** to draw another card
4. Try to reach a total of **21** without going over
5. Game results:

   * 🎉 21 → You win (Blackjack)
   * 💥 >21 → You lose
   * 🤔 <21 → Continue playing

---

## 📊 Game Rules

* Number cards = face value
* Face cards (J, Q, K) = 10
* Ace = 1 or 11
* Dealer stands at 17
